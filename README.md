# tool_01_laravel-new-project
Laravel 12 + Breeze の新規プロジェクトを macOS 上で自動構築するセットアップスクリプト。PHP / Composer / Node のチェック、環境自動構成、Breeze インストール、.env 初期化、APP_KEY 生成までを一括で実行します。 Auto setup script for creating Laravel 12 + Breeze projects on macOS. Automatically checks dependencies, installs Breeze, prepares .env, and configures the project in one command.
